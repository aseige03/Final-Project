3) I created a table called superbowl_champions and in this table the following columns were included: game, date_season, winning_team, score, losing_team, venue, and city. Using these values in an SQLite query I was able to extract the csv file called "game_outcome.csv" this csv was used to make the interactive web applcation you will see on the next page.

Example query:

SELECT game, date_season, winning_team, score, losing_team, venue, city FROM superbowl_champions;

4) On the next page using the arrows at the bottom of the screen you will find my interactive web application. In this web application you as the user you will be able to see all of the following: The game score - winner in green loser in red, The teams that played in that superbowl and their logos, what superbowl it was in roman numerals, the date of the game, and where the game was held. In order to interact with the application there is a drop bar located on the right hand side of the score board where you will select the year and it will show all of the data for that game.
